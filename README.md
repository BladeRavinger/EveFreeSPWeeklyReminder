# EveFreeSPWeeklyReminder

Simple App to Set a system tray reminder, it reminds you to login and claim your free SP

## Usage

Grab Latest release, put in a nice permanent folder like Documents. 
Open the file, Done.

A New system tray icon will, appear, Set `Start with Windows` and forget.

`Stop and Exit` will stop timers and exit the app
`Hide Tray Icon` will hide the tray icon until you start the app again/restart system but leave timers running
`Uninstall` will remove any temp files, unckeck `Start with Windows` and delete the exe to remove all traces of the program.
`Force trigger` as you would guess, forces the timer to trigger now, and waits for the game to launch before resetting.

## Remove

unckeck `Start with Windows`, click `Uninstall / Remove`, delete the exe
